# SORA2

SORA2 is an open-source project designed to interact with AI models such as ChatGPT.  
(SORA2は、ChatGPTなどのAIモデルと連携するオープンソースプロジェクトです。)

## Features / 主な機能

- Seamless integration with ChatGPT and other AI models  
  (ChatGPTなどのAIモデルとのシームレスな連携)
- Fast and lightweight
- Easy to use API

## Usage / 使い方

1. Install dependencies  
2. Set up your API keys  
3. Run the application
